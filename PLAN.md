# 极简几何参数风 — 艺术交互网站

## 技术栈
- **纯原生** HTML/CSS/JS + Canvas 2D（零依赖，极致性能）
- 无框架、无构建工具，浏览器直接打开即用

## 项目结构
```
art-interactive/
├── index.html          # 主页面
├── css/
│   └── style.css       # UI 覆盖层样式
├── js/
│   ├── main.js         # 入口：初始化、动画循环
│   ├── palette.js      # 莫兰迪色系生成器
│   ├── geometry.js     # 正多边形、对称纹样生成
│   ├── particles.js    # 圆形/矩形粒子系统
│   └── easing.js       # 平滑缓动函数库
└── README.md
```

## 核心功能（MVP）

### 视觉层 (Canvas)
- **网格布局**：全屏均匀网格，每个单元格渲染一个几何图形
- **几何元素**：正多边形（3-8边）、圆形、圆角矩形
- **对称纹样**：单个图形内部镜像/旋转对称
- **线条渐变**：描边使用线性/径向渐变
- **莫兰迪色系**：低饱和、柔和色调自动生成

### 交互层
- **鼠标悬停**：图形放大 + 颜色变亮 + 缓慢旋转
- **点击**：切换图形类型（多边形↔圆形↔矩形）
- **全局缓动**：所有过渡使用 easeInOutCubic

### 粒子层
- 背景漂浮圆形/矩形小粒子，缓慢移动
- 鼠标吸引/排斥粒子
