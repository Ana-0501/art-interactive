# ArtFilter — 图像艺术处理

拖放上传图片，选择 6 种艺术效果，即时预览并导出。

## 六种效果

| 效果 | 说明 |
|------|------|
| 🔴 **点彩重构** | 用彩色圆点重建图像（修拉画派） |
| 🔷 **几何解构** | 三角/菱形剖分填色（Low-poly） |
| 🖋️ **水墨轮廓** | 边缘提取 + 宣纸底色（素描水墨风） |
| 👾 **像素复古** | 降采样像素化（8-bit 游戏风） |
| 🟡 **波普网点** | 半色调圆点（漫画印刷感） |
| 🎨 **色彩抽离** | 保留主色调，其余去色（海报风） |

## 技术

- 纯前端 Canvas 2D 像素级处理
- 零依赖，浏览器直接打开
- 图片不上传服务器，本地处理

## 使用

```bash
open index.html
# 或直接双击 index.html
```

1. 拖放/点击上传图片
2. 选择效果
3. 调节强度滑块
4. 点击「导出图片」
